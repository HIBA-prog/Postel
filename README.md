# Postel
PostEL une application web de blogging innovante centrée exclusivement sur les hôtels. 
=======
PostEL
PostEL est une application web de blogging dédiée aux hôtels, conçue pour permettre aux utilisateurs de partager leurs expériences et avis sur différents établissements. 

Fonctionnalités
Création d’un compte utilisateur et authentification sécurisée. 🔒
Ajout et gestion des publications avec des avis d’hôtels. 🏨
Section de commentaires pour favoriser les échanges. 💬
Recherche par localisation ou catégorie d’hôtel. 🔍
Interface responsive pour un affichage optimal sur mobile et desktop. 📱💻

Technologies utilisées
Front-End :

React.js
HTML, CSS, Bootstrap, Tailwind CSS
Back-End :

Node.js, Express.js

Base de données :

MongoDB
Outils complémentaires :

Git & GitHub pour la gestion de version
Figma pour le design UI
